# PropertyIQ

Intelligent Property Analysis Platform for Irish Real Estate

## Features

- Comprehensive property analysis for Irish real estate
- AI-powered insights using GPT-4o
- PropertyRegister.ie CSV data integration (2010-2025)
- Enhanced commute time analysis
- Search history with database storage
- Stripe subscription integration

## Setup

1. Install dependencies: `npm install`
2. Set up environment variables
3. Run database migrations: `npm run db:push`
4. Start the application: `npm run dev`
